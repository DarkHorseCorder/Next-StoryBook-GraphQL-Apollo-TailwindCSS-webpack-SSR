# Typography

The following tokens are available as Tailwind utilities for text styling:

- `text-h1`
- `text-h2`
- `text-h3`
- `text-h4`
- `text-h5`
- `text-h6`
- `text-body-lg`
- `text-body-lg-heavy`
- `text-body-md`
- `text-body-md-heavy`
- `text-body-sm`
- `text-body-sm-heavy`
